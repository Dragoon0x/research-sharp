# research-sharp

Research craft and intelligence — 557 skills and 35 commands across market research, user research, competitive intelligence, data analysis, survey design, qualitative methods, research strategy, and product research.

## What this is

An open-source skill pack for AI coding agents and research practitioners. Every skill has a practitioner-level depth of knowledge — real frameworks, real case studies, real mental models, and real execution guidance.
